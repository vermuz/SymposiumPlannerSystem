This distribution just needs users to do a little bit configuration when install SymposiumPlanner 2011.

1. Make sure JDK is intalled and its environment variables are available.
2. Make sure Apache Tomcat is installed and its its environment variable is available.
3. Please make sure the port 8888, 8080, 9990, 9993, 9998 are available.
4. Download the package from the Sourceforge at: https://sourceforge.net/projects/sps2011/files/
5. Unzip the package.
6. Move all folders except "SPS20110916" to %TOMCAT%/webapps/.
7. Start Apache Tomcat.
8. In folder "SPS20110916", run SymposiumPlanner 2011 server.
